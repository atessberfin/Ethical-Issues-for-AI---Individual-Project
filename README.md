# Recipe Recommendation Fairness Project

This repository contains the code for detecting and mitigating popularity and cuisine representation bias in a recipe recommendation system. 

### Key Features
- Popularity normalization
- Fair re-ranking for cuisine diversity
- Analysis of bias before and after mitigation
- Implemented in Python / Google Colab

### Structure
- `notebooks/` — main project notebook
- `data/` — dataset reference (not included)
- `report/` — final report (PDF)
